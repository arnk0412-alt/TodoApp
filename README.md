# TodoApp 📝

A simple and clean Todo application built with React Native and Expo.

## Features

- ✅ Add new todos
- ✏️ Edit existing todos
- 🗑️ Delete todos
- ☑️ Mark todos as completed
- 🔍 Search todos
- 🔽 Filter by All / Completed / Pending
- 🌙 Dark mode toggle
- ✨ Smooth add/delete animations
- 💾 Data persisted using AsyncStorage

## Tech Stack

- React Native (Expo)
- TypeScript
- AsyncStorage

## Setup & Installation

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or above)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [Expo Go](https://expo.dev/client) app on your Android/iOS device

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/arnk0412-alt/TodoApp.git
   cd TodoApp
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Install AsyncStorage**
   ```bash
   npx expo install @react-native-async-storage/async-storage
   ```

4. **Start the development server**
   ```bash
   npx expo start
   ```

5. **Run on device**
   - Scan the QR code using the Expo Go app on your phone
