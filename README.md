\# TodoApp 📝



A simple and clean Todo application built with React Native and Expo.



\## Features



\- ✅ Add new todos

\- ✏️ Edit existing todos

\- 🗑️ Delete todos

\- ☑️ Mark todos as completed

\- 🔍 Search todos

\- 🔽 Filter by All / Completed / Pending

\- 💾 Data persisted using AsyncStorage



\## Tech Stack



\- React Native (Expo)

\- TypeScript

\- AsyncStorage

\## Setup \& Installation



\### Prerequisites



Make sure you have the following installed:



\- \[Node.js](https://nodejs.org/) (v16 or above)

\- \[Expo CLI](https://docs.expo.dev/get-started/installation/)

\- \[Expo Go](https://expo.dev/client) app on your Android/iOS device



\### Steps



1\. \*\*Clone the repository\*\*

&#x20;  ```bash

&#x20;  git clone https://github.com/YOUR\_USERNAME/TodoApp.git

&#x20;  cd TodoApp

&#x20;  ```



2\. \*\*Install dependencies\*\*

&#x20;  ```bash

&#x20;  npm install

&#x20;  ```



3\. \*\*Install AsyncStorage\*\*

&#x20;  ```bash

&#x20;  npx expo install @react-native-async-storage/async-storage

&#x20;  ```



4\. \*\*Start the development server\*\*

&#x20;  ```bash

&#x20;  npx expo start

&#x20;  ```



5\. \*\*Run on device\*\*

&#x20;  - Scan the QR code using the Expo Go app on your phone
